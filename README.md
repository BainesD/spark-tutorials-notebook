# How To Use This Repo

- Go to https://mybinder.org and input this repos address (https://github.dev/BainesD/spark-tutorials-notebook) into the box labeled "GitHub repository name or URL"
- Click the "launch" button
- Once the binder has launched open a new terminal from the launcher and run `pip install -r requirements.txt`
- Once the command finishes you can run pyspark from python file or jupyter notebook file
